# 📦 UApiPro-Toolbox 工具箱插件 

本插件是为 **AstrBot** 机器人框架开发的综合功能包。它通过集成 [UApiPro](https://uapis.cn) 提供的开放接口，为机器人提供精美的图文交互体验。在将来也会适配更多接口，尽请期待。
欢迎提issue或pr增加功能。


## 💡 核心功能

- **双模式显示**：默认渲染为 HTML 图片卡片；可在设置中切换为文本模式，适配低流量环境。
- **LLM 工具调用**：16 项查询指令同时注册为 function calling 工具，LLM 可根据对话内容自动调用。
- **安全防护**：注入阻断：所有输入经转义与长度限制，防止 XSS 与命令注入。
- **磁盘零堆积**：临时文件在图片发送后立即删除，不占用服务器磁盘空间。

## 📂 项目结构

插件采用**转接器模式**设计，实现了逻辑与展示的解耦：

```text

├── llm_tools.py        # LLM function calling 工具注册
├── _conf_schema.json   # 配置
├── card_renderer.py    # 渲染
├── metadata.yaml       # 插件元数据
├── apis/               # API 矩阵
│   ├── weather.py      # 天气查询模块
│   ├── ipquery.py      # IP 归属地查询模块
│   ├── mcquery.py      # Minecraft 服务器状态查询
│   ├── hitokoto.py     # 一言/语录获取
│   ├── news.py         # 每日新闻摘要获取
│   ├── random_img.py   # 随机图片库获取
│   └── ...             # 其他 API 模块
├── hotboard/           # 全网热榜聚合模块
│   ├── __init__.py     # 统一入口
│   ├── fetcher.py      # 请求调度与错误处理
│   ├── renderer.py     # 热榜专属 HTML 渲染引擎
│   ├── bilibili.py     # 哔哩哔哩解析器
│   ├── acfun.py        # A站解析器
│   ├── hellogithub.py  # HelloGitHub 解析器
│   ├── netease_music.py# 网易云音乐解析器
│   ├── qq_music.py     # QQ音乐解析器
│   ├── weread.py       # 微信读书解析器
│   └── icons/          # 各平台图标
```

## ⚙️ 插件设置说明

您可以在 AstrBot Web 面板的`插件配置`中进行全中文可视化设置：

*   **接口密钥 (Token)**: 填入您在 UApiPro 后台获取的专属 Token（可留空）。
*   **随机图片库范围**: **多选下拉框**。支持勾选 `ACG`、`风景`等分类。机器人会仅从您勾选的范围中随机挑选。
*   **指令冷却时间**: 自定义触发间隔（默认 5.0 秒），防止并发刷屏。
*   **定时早报配置**:
    *   **推送开关**: 开启后每天自动执行推送任务。
    *   **具体时间**: 格式为 `HH:MM`。
    *   **推送名单**: 支持多行输入群号或 QQ 号（直接填纯数字即可）。
